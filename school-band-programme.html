Programme foundations

What it is

Pupils learn together as a band in a safe, structured, confidence-building environment. The programme is beginner-focused, built for consistency, and designed to run smoothly inside a school timetable.

Group structure

  • Up to 4 pupils per band
  • Minimum 3 pupils to run
  • Beginner-focused for primary age pupils

Instruments provided

  • Keyboard
  • Electric guitar
  • Electric bass
  • Electronic drums

Non-negotiables

Key rules

  • Fixed weekly slot: same day/time each week.
  • One instrument per term: choice stays fixed for the whole block.
  • Allocation balances the band: preferences are considered, balance is priority.
  • No rearranging sessions for individual absences.
  • Behaviour & safeguarding always comes first.

Performance included: one informal performance happens during the final session of each block.
